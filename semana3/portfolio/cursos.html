A Labenu é uma startup que forma desenvolvedores Web Full Stack e prepara para o mercado de trabalho. Com uma carga horária de mais de 1000 horas de programação, o período de curso contempla não só aulas completas com desafios e exercícios diários, mas também palestras de soft skills, how to learn e atualizações sobre o mercado de tecnologia.

O curso apresenta conceitos, ferramentas e projetos diários de Frontend ( Html, CSS, Javascript, React e Jest) e Backend ( Node, Typescript, MySQL, AWS e Firebase).
